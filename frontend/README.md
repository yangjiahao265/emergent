# Northstar Data Portal — Frontend

基于 React 19 + Tailwind CSS 的前端应用。

## 开发

```bash
yarn install
yarn start
```

## 构建

```bash
yarn build
```

## 目录说明

```
src/
├── App.js              # 应用入口组件
├── App.css             # 全局样式
├── index.js            # React 挂载点
├── index.css           # Tailwind 基础层
├── components/ui/      # Radix + shadcn/ui 组件库
├── hooks/              # 自定义 Hooks
└── lib/                # 工具函数
```
